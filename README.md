## 相关说明

OS: ubuntu 18.04

## 实验一大致流程
Step1. 编写路由器程序并编译  
Step2. 运行router_topo.py, 此时会自动进入mininet交互界面  
Step3. 执行xterm r1命令, 会新开启一个的窗口, 可以理解为进入router1的系统, 在此执行./router, 路由器启动完成  
Step4. 执行xterm h1, 进入host1, ping相应的地址
